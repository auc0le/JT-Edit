JT-Edit - Keyboard Shortcuts & Help

                    
                

                

                    

                        
                        

                            
                            
                        

                    

                    

                        

                            
File Operations

                            

                                

                                    Ctrl+O
                                    Open file (JT or PNG)
                                

                                

                                    Ctrl+S
                                    Save file
                                

                            

                        


                        

                            
Animation Controls

                            

                                

                                    Space
                                    Play/Pause animation
                                

                                

                                    ←/→
                                    Previous/Next frame
                                

                                

                                    Ctrl+N
                                    Add new frame
                                

                                

                                    Ctrl+D
                                    Clone current frame
                                

                                

                                    Delete
                                    Delete current frame
                                

                                

                                    Escape
                                    Stop animation
                                

                            

                        


                        

                            
Transform & Movement

                            

                                

                                    Shift+↑/↓/←/→
                                    Shift all pixels in direction
                                

                            

                        


                        

                            
Zoom & View Controls

                            

                                

                                    +/=
                                    Zoom in
                                

                                

                                    -
                                    Zoom out
                                

                                

                                    F
                                    Fit to screen
                                

                                

                                    Ctrl+Shift+A
                                    Toggle auto scaling
                                

                            

                        


                        

                            
Color & Paint Tools

                            

                                

                                    X
                                    Swap foreground/background colors
                                

                                

                                    Shift+F
                                    Fill with foreground color
                                

                                

                                    Shift+B
                                    Fill with background color
                                

                                

                                    Ctrl+Shift+C
                                    Toggle color format (3-bit/24-bit)
                                

                            

                        


                        

                            
Mode & Canvas

                            

                                

                                    Ctrl+M
                                    Toggle static/animation mode
                                

                                

                                    1-5
                                    Quick canvas size selection
                                

                            

                        


                        

                            
Help & Interface

                            

                                

                                    F1
                                    Show this help dialog
                                

                                

                                    ?
                                    Show this help dialog
                                

                            

                        


                        

                            
Drawing

                            

                                

                                    Left Click
                                    Paint with foreground color
                                

                                

                                    Right Click
                                    Paint with background color
                                

                                

                                    Click + Drag
                                    Paint continuously while dragging
